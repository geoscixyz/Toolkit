    \section{2.2.1. Magnetic Data Response and
Visualization}\label{magnetic-data-response-and-visualization}

\subsection{Magnetic response over the Search Phase II project
area}\label{magnetic-response-over-the-search-phase-ii-project-area}

Gridded residual magnetic data from Geoscience BC's Search Phase II
project (Anomalous Magnetic Field) were downloaded from Geoscience BC's
website. The magnetic data is interpolated onto a 50 \(m^2\) grid. The
data are shown using coordinate system NAD 83 UTM Zone 9.

    \subsection{Define a project area}\label{define-a-project-area}

Let's choose a subset of data to view more closely. The map on the left
shows the full Search Phase II magnetic dataset with a movable 'window'
outlined in black, and the map on the right shows just the data
encompassed by the window. The window can be resized and moved anywhere
over the data using the slider bars. The default window is 40 km x 40 km
in size, a size typical of surveys flown to explore magnetic responses
